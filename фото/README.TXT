Данный репозиторий состоит из одностраничного сайта "Яндекс.Такси" (с наличием рабочей AJAX-формы, написанной на языке программирования JavaScript), запускаемого (с применением локального сервера OSPanel) 
через файл index.html, который после нажатия по кнопке "Отправить" пересылает текстовые данные из двух полей ввода внутрь файла server.php, где эти данные скачиваются на ПК (внутрь папки данного сайта, где 
помимо файлов index.html и server.php хранится папка с фотографиями) в виде файла forma.txt и отсылаются на указанный внутри файла server.php адрес электронной почты.
